# Customer Engagement & Product Utilization Analytics

## Project Overview

This project analyzes customer retention using engagement behavior, product utilization, relationship strength, and financial commitment indicators.

The objective is to identify customer behaviors associated with churn and provide actionable retention strategies for banking organizations.

## Features

* Customer Retention KPIs
* Engagement Retention Analysis
* Product Utilization Analysis
* High-Value Disengaged Customer Detection
* Salary vs Balance Risk Analysis
* At-Risk Premium Customer Identification
* Relationship Strength Assessment
* Credit Card Stickiness Analysis
* Sticky Customer Profiling
* Executive Insights Dashboard

## Technologies Used

* Python
* Streamlit
* Pandas
* Plotly
* Excel

## Dataset Fields

* CustomerId
* Geography
* Gender
* CreditScore
* Age
* Balance
* NumOfProducts
* HasCrCard
* IsActiveMember
* EstimatedSalary
* Exited

## Dashboard KPIs

* Customer Base
* Churn Rate
* Active Customers
* Average Products per Customer
* Engagement Retention Ratio
* Product Depth Index
* Relationship Strength Index
* Credit Card Stickiness Score

## Run Application

python -m streamlit run app.py

## Project Outcome

The analysis demonstrates that engagement and product adoption are stronger predictors of retention than financial strength alone. The dashboard enables proactive identification of churn risk and supports engagement-driven retention strategies.
